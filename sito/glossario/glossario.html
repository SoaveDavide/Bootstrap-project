INFORMATICA SISTEMI SOCKET GLOSSARIO
Socket

Interfaccia per la comunicazione di dati tra processi attraverso una rete. Utilizzate per implementare protocolli come TCP e UDP.

Leggi di più
DBMS

Software che gestisce l'accesso ai dati nei database. Garantisce integrità, sicurezza e concorrenza delle operazioni.

Leggi di più
Three-Way Handshake

Procedura in tre fasi (SYN, SYN-ACK, ACK) utilizzata per stabilire connessioni TCP sicure tra client e server.

Leggi di più
File System

Un file system è una componente fondamentale di un sistema operativo che gestisce la memorizzazione, l'organizzazione, il recupero e la manipolazione dei dati su dispositivi di archiviazione come dischi rigidi, SSD, unità USB e altri supporti di memoria.

Leggi di più
Porta

Numero utilizzato per distinguere applicazioni in comunicazione su una rete. TCP/UDP utilizzano porte per gestire connessioni specifiche.

Leggi di più
Database

Un database è una raccolta organizzata di dati strutturati, gestiti in modo da consentire l'accesso, la manipolazione e la gestione efficienti delle informazioni. I database vengono utilizzati in vari contesti, come applicazioni aziendali, sistemi informatici, siti web e software, per archiviare informazioni in maniera sicura e accessibile.
